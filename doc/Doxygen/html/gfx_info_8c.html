void gfxInfo_initMatrix ( Binary *  base,
uint8  size 
)

Función que genera la Matrix de manera independiente Para poder construir los sprites asociados a cada elemento de la matriz. Se encarga de reservar memoria en el banco de memoria de la NDS y, una vez obtenido el puntero al sprite, se vincula a su posición correspondiente en la matriz.

Parameters
baseDirección de memomria del primer elemento de la matriz bidimensional.
sizeNúmero de filas/columnas (matriz cuadrada).

@nota: *(base + i*size + j) En memoria dinámica, C trata los arrays bidimensionales como si fueran un vector, con lo que para acceder a cada elemento lo hago mediante la desreferencia de la posición dada por: dirección base + número de fila*número de elementos en esa fila

  • número de columna. De ésta manera puedo comprobar el id del GFX en esa posición.
