Detailed Description

Fichero en el que se definen las funciones encargadas de mostrar en la pantalla de arriba el texto oportuno. Emulando una "User Interface" con simplemente texto y símbolos. De ahora en adelante se les llamará Menús o UI.

Author
Geru-Scotland.

Ésta manera no es que esté muy bien, pero durante el desarrollo he encontrado varios bugs con la gestión de libnds y su consola, así que he decidido el hacerlo de manera simple y, bastante troglodita.
